# ColorPages - Personal project. 
# Don't forget to add the pdf file.
This repo contains code to find color pages in a .pdf file. 

It prints the color Figures present in the pdf file - along with their caption. 

Typical use case: Useful to find out which pages are monochrome and which of them are colored during pritnouts. 

# usage:
python findColorPages.py

# output:
1. Prints the page numbers on the console
2. Creates the file figure_pages.txt with captions of the figures.
